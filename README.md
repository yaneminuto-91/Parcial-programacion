# Parcial-programación:



**#Proyecto Casa De Cambio-Yanne.Minuto**



Sistema tecnológico desarrollado en lenguaje Java y Pseint para la conversión y registro de transacciones en  casa de cambio Yanne.Minuto.



Presentado como parcial Final de la asignatura POO con NRC: 80847





---

**# Objetivo del Proyecto**



Generar una solución tecnológica capaz de:



* Realizar conversiones entre COP,USD y EUR. 
* Simular montos antes de registrar la transacción 
* Registrar operaciones simuladas en el aplicativo 
* Visualizar y borrar el historial de transacciones 
* Aplicar conceptos fundamentales en POO ( clases, objetos, métodos, atributos, Herencia simple, cardinalidad, Encapsulamiento get-set, polimorfismo,  abstracción)
* Desarrollar toda la documentación implícita para la implementación de proyectos tecnológicos (pseudocódigo, diagrama de flujo, Wireframe, lenguaje java, requerimientos funcionales y no funcionales)
